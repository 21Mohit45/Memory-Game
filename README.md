# 🎮 Memory Game

A simple, responsive Memory Card Game built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Splash screen
- Player ID (1–4 letters/numbers)
- Easy (4×4)
- Hard (6×6)
- Timer
- Moves counter
- Card flip animation
- Top 5 leaderboard (localStorage)
- Reset leaderboard
- Restart / Home / Play Again
- PWA support (installable)
- Fully responsive

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Service Worker (PWA)

---

## 📱 How to Run

### Option 1: Local
Just open `index.html` in browser.

### Option 2: VS Code
Use Live Server extension.

---

## 🌐 Deploy on Vercel
